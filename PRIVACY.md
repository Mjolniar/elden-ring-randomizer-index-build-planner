# Privacy Policy

Elden Ring Randomizer Index and Build Planner is an offline desktop utility.

## Data Collection

The app does not collect telemetry, analytics, personal information, or usage data. It does not send spoiler logs, custom builds, favorites, acquired-item state, or other local app data to the developer or to any external service.

## Local Files

The app reads spoiler log files only when the user selects or drops them into the app. It stores local app state on the user's computer, including cached spoiler logs, favorites, acquired-item tracking, and custom build data.

## Network Access

The app has no server backend. Item wiki links open in the user's default browser only when clicked by the user.

## Game Files

The app does not modify Elden Ring files, inject into the game, read game memory, or interact with the running game process.
